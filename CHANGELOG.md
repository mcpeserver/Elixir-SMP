# Changelog — OZI STAR SMP Website

Semua perubahan penting pada proyek website ini akan didokumentasikan di file ini.

## [1.0.0] - 2026-07-06
### Added
- Inisialisasi dasar web server OZI STAR SMP menggunakan React, Vite, dan Tailwind CSS.
- Topbar Developer dinamis yang terintegrasi penuh dengan fetch API GitHub Ran Dev.
- Hero Banner dengan logo pixel art custom dan background padang rumput bertema Minecraft.
- Server Status interaktif dan tombol salin IP otomatis.
- Tampilan 3D Kepala Skin Staf dengan integrasi Minotar.
- Leaderboard Donatur dengan visualisasi kartu donatur emas, perak, perunggu.
- Aturan (Rules) Server yang dibagi menjadi dua kolom (Diperbolehkan vs Tidak Diperbolehkan).
- Jam PvP bebas berbasis timeline horizontal.
- Tombol tautan media sosial komunitas (WhatsApp, Discord, YouTube, TikTok).
- SEO Friendly Metadata, Schema JSON-LD, Sitemap, dan file Robots.txt.
